Bắt Đầu

Chào mừng bạn đến với tài liệu hướng dẫn của Thazh Labs. Tài liệu này sẽ giúp bạn làm quen với các sản phẩm và dịch vụ của chúng tôi.

Yêu Cầu Hệ Thống

  • Hệ điều hành: Windows 10+, macOS 10.14+, hoặc Linux
  • Bộ nhớ: Tối thiểu 4GB RAM
  • Không gian lưu trữ: 500MB trống

Cài Đặt

Hướng dẫn cài đặt các sản phẩm của chúng tôi.


// Ví dụ lệnh cài đặt
npm install thazh-package
// hoặc
pip install thazh-package
